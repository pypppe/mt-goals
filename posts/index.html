1
Create a free Supabase project

Go to supabase.com and sign up for free. Then click "New project". Give it any name you want, set a password, pick any region, and click "Create new project". Wait about 30 seconds for it to finish.

2
Create the posts table

In your Supabase project, click "SQL Editor" in the left sidebar. Copy the code below, paste it in, and click "Run".

create table posts ( id bigserial primary key, display_name text not null, content text not null, created_at timestamptz default now() ); alter table posts enable row level security; create policy "read all" on posts for select using (true); create policy "insert all" on posts for insert with check (true);
3
Enter your Project URL
